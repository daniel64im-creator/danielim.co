Instant FTE Lookup for SDR Prospecting

Prospecting Workflow Acceleration Tool

Operational lookup system designed to surface institution size (FTE) instantly during prospecting, reducing the need for manual research and improving prioritization speed for SDR workflows.

System Overview

SDR prospecting frequently required understanding institutional size in order to prioritize outreach and determine account fit. While this data existed in external sources, retrieving it during active prospecting required switching between multiple research tools and websites.

This project introduced a fast-reference lookup mechanism that allowed SDRs to retrieve FTE information directly within their working environment.

Structural Gap

  • FTE data required external searches during prospecting.
  • Manual research slowed SDR workflow execution.
  • Account prioritization lacked consistent size signals.
  • Repeated lookups created redundant effort.

Architecture

  • Embedded Lookup System - Central dataset containing institution FTE values.
  • Search-Based Retrieval - SDRs could retrieve size data instantly during targeting.
  • Standardized Output Format - Consistent formatting allowed reps to quickly interpret results.
  • Workflow Integration - Designed to minimize clicks and avoid interrupting prospecting flow.

Tools

  • Google Sheets dataset
  • Institution enrollment data sources
  • Internal SDR workflow sheets

Scale

  • Primary users: SDR prospecting team
  • Use frequency: every prospecting session
  • Purpose: accelerate account qualification

Operational Impact

  • Reduced research time during prospecting.
  • Improved consistency of account prioritization.
  • Enabled faster targeting decisions.
  • Reduced repeated manual data lookups.

Future Improvements
